PizzaShop Front End

##Setup
Downlaod Node.js:

    https://nodejs.org/en/download/
Install npm

    npm install

Install Node

    npm install node

Install JQuery

    npm install jquery

##Build
To build this project with npm use:

    npm run-script build

##Run Project
The following command will run the Project:

    npm run start

Use `CTRL + C` to stop.

##To Run Locally

    https://localhost:3000
